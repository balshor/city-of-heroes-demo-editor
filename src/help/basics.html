What is a demo file?

A demo file is a specially formatted text file containing instructions for the City of Heroes client to show a predetermined sequence using the game's renderer, models, maps, etc. These files are usually generated by the client to record actual in-game events. However, there is no particular reason why demo files cannot instruct the client to show things that did not or cannot happen in the game world.

Don't think of a demo file as a movie. A better analogy would be to think of it as a very detailed script that tells a producer (the game engine) which actors to use, how to costume them, where they should stand and move, what they should say, and what sort of actions they should perform.

How do I record a demo file?

Use the slash command /demorecord [filename] in game to start recording a demo. (Here, replace [filename] with the name of the file you wish to save your demo as. The extension .cohdemo will automatically be appended if none is specified.) To stop recording, use /demostop. Demo files will be saved in a subfolder called client_demos of your main City of Heroes folder.

How do I play a demo file?

There are a couple different options for this:

  • To play a demo, create a new shortcut to the City of Heros program, right-click on it, and select properties. In the target box, after whatever is already there, add -demoplay (outside the quotes). You should now be able to drag-and-drop the demo file onto this shortcut to play it. To stop playback, press ESC.
  • You can use a demo launcher, which will automatically pass in the necessary parameters to the City of Heroes client. This program can also launch demos, using the launch button in the toolbar.
